# npcfinder-game

`npcfinder-game` is a small, relaxing 2D social game for `npcfinder.com`, focused on cozy home customization and low-pressure friend visits in a calm, expressive world.

## Tech Stack

- Phaser 3
- Vite
- TypeScript

## Commands

- `npm run dev`
- `npm run build`
- `npm run preview`
